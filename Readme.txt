This is a Premium Plugin. Pit Login Welcome is a Plugin which can give a Random Welcome message to your blog/Website's users when login. You can also change WordPress login page Logo & Background.

<strong>Background Change:</strong> Login page background color or photo can be change from here. So your website login page will be more beautiful & nice.

<strong>Login Page Message:</strong> This option is first time in WordPress. You can some messages like welcome messages or tips & tricks for your registered users. So anyone open the login page she/he can read your welcome messages or tips & tricks. All messages show randomly. You can add unlimited messages.

== Installation ==

= From WordPress Plugin Directory =
* Search 'Pit-Login-Welcome' to Plugins library
* Install the plugin by click the Install option
* Activate the plugin
* Setup the plugin from settings page under your WordPress dashboard.

= From Your Computer =
* Download the 'Pit-Login-Welcome.zip' file from WordPress plugins directory
* Upload to '/wp-content/plugins/' directory
* Unzip the plugin folder
* Activate the plugin through the 'Plugins' menu in WordPress Dashboard
* Setup the plugin from settings page under your WordPress dashboard

= by FTP =

* Download the plugin from WordPress Plugins directory
* Unzip the zip folder
* Upload all folders & files to '/wp-content/plugins/' directory
* Activate the plugin
* Setup the plugin from settings page under your WordPress dashboard
